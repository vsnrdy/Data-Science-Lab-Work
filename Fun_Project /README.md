# Number Guessing Game

## Description

This is a simple Python game where the computer randomly selects a number between 1 and 100, and the user has to guess it within limited attempts.

## Dataset

* No dataset required
* Random number generated using Python

## Steps Performed

1. Generated a random number
2. Took user input
3. Compared input with actual number
4. Provided hints (high/low)

## Results

* User can guess the number with feedback
* Improves logical thinking

## Tools Used

* Python
* Random module

## Conclusion

This project demonstrates the use of loops, conditions, and user input in Python.
