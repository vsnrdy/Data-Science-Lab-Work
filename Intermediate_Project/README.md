# EV Battery Health Prediction using Machine Learning

## Description

This project focuses on predicting the health of electric vehicle (EV) batteries using machine learning techniques. The goal is to analyze battery parameters such as voltage, current, temperature, and charge cycles to estimate the State of Health (SOH) of the battery.

## Dataset

* Source: NASA Battery Dataset / Kaggle EV datasets
* Description: The dataset includes features like voltage, current, temperature, cycle count, capacity, resistance, and charging time.

## Steps Performed

1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Visualization of battery performance
4. Model Building using Regression techniques

## Results

* Identified patterns in battery degradation
* Predicted battery health using machine learning models

## Tools Used

* Python
* NumPy, Pandas, Matplotlib, Seaborn, Scikit-learn

## Conclusion

Machine learning techniques can effectively predict battery health and improve battery management systems by identifying degradation patterns and optimizing performance.
